## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

This page is meant to serve as a showcase for my experience with computer programming. It includes screenshots of some of my projects, along with links to their deployments and repositories. I've also included a short bio, links to my LinkedIn and GitHub, and a downloadable resume.

### Built With

* [MDB5](https://mdbootstrap.com/)
* Vanilla HTML, CSS, and Javascript



<!-- GETTING STARTED -->
## Getting Started

Once you've landed on the home page, scroll down for more information. The carousel part way down the page cycles through screenshots of my projects. Each has clickable links. Click the document icon to download a copy of my resume.

### Prerequisites

This page should be compatible with all browsers, and is mobile responsive.
